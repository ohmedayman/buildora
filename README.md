# Buildora - منصة بناء صفحات الويب

منصة بناء صفحات ويب احترافية بدون أي خبرة برمجية. اسحب وأفلت العناصر، خصّص التصميم، وانشر صفحتك مباشرة.

## المميزات

- بناء صفحات بأسحب وأفلت
- 12 قالب جاهز
- 25+ عنصر جاهز
- دعم RTL للعربية
- تجربة مستخدم متجاوبة
- SEO متقدم
- إحصائيات زوار حية
- subdomain مجاني لكل مستخدم

## التقنيات

- Next.js 16
- React 19
- Tailwind CSS 4
- Supabase (Auth + Database)
- TypeScript

## التشغيل المحلي

```bash
npm install
npm run dev
```

## البيئة

انسخ `.env.example` إلى `.env.local` وأضف المتغيرات:

```bash
cp .env.example .env.local
```

## النشر على Vercel

1. ارفع المشروع على GitHub
2. اربط GitHub repo بـ Vercel
3. أضف متغيرات البيئة في Vercel Dashboard
4. النشر تلقائي
