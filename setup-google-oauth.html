🔧 إعداد Google OAuth لـ Buildora

اتبع الخطوات بالترتيب — كل حاجة جاهزة تنسخها

1 افتح Google Cloud Console
فتح Google Cloud Console ↗
2 اضغط + Create Credentials → OAuth client ID
من الأعلى اضغط على الزر الأزرق
3 املأ البيانات
Application type: اختار Web application
Name: اكتب Buildora
4 أضف Authorized redirect URI
اضغط + Add URI والصق الرابط ده:
https://scosuziupwgxxjmildnr.supabase.co/auth/v1/callback
اضغط على الكود عشان يتنسخ تلقائي
5 اضغط Create
هتطلع نافذة فيها Client ID و Client Secret
6 ابعتلي Client ID و Client Secret
انسخهم من النافذة وأعتليهم هنا — وأنا أكمّل الباقي في Supabase تلقائي!
💡 ملاحظة: لو مش لاقي Credentials، اضغط على القائمة الشمال → APIs & Services → Credentials
